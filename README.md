# 巧虎手偶 LP

台湾向け・0〜2歳の家庭に向けた「巧虎手偶」紹介ランディングページ。
LINE配信後に見てもらう商品紹介ページとして、動画 → 理解 → 購入までを1ページの縦スクロールで完結させる。

公開先は購入ページ（Benesse公式ストアの巧虎手偶ページ）へ誘導する。

---

## ファイル構成

```
.
├── index.html              … LP本体（HTML/CSS/JSを1ファイルに内包）
├── qiaohu_fv.mp4           … ファーストビューの動画
├── poster.jpg              … 動画のポスター（読み込み中・自動再生不可時に表示）
├── favicon.png / .ico / favicon-32.png / apple-touch-icon.png … サイトアイコン
├── images/
│   ├── why1-mom-baby.jpg       … WHY① ママ＋巧虎＋赤ちゃん
│   ├── why2-tiger-closeup.jpg  … WHY② 巧虎の色の特写
│   ├── why3-family.jpg         … WHY③ 三世代の家族
│   ├── stage1-comic.jpg        … 0〜6個月ステージの遊び方イラスト
│   ├── summary-hug.jpg         … まとめ（子供が巧虎を抱きしめる）
│   ├── product.jpg             … 購入セクションの商品写真
│   └── qiaohu-peek.png         … 家族セクションで覗く巧虎（透過PNG）
├── netlify.toml            … Netlify公開設定（ビルド不要・直下を公開）
├── .gitignore
└── README.md
```

ビルド工程はなし。`index.html` を開けばそのまま動く静的サイト。

## よくある編集

### テキストを直す
`index.html` を開いて該当の文言を書き換えるだけ。

### 写真を差し替える
`images/` の中の同名ファイルを差し替えれば、HTMLの変更は不要。
（例：`images/why1-mom-baby.jpg` を新しい画像で上書き）

### 動画を差し替える
`qiaohu_fv.mp4` を差し替える。あわせて `poster.jpg`（動画の1コマ）も更新するとよい。

## デプロイ（Netlify）

このリポジトリを Netlify に Git 連携すると、`main` ブランチへ push するたびに自動で公開される。

1. Netlify → Add new site → Import from Git → このリポジトリを選択
2. Build command は空、Publish directory は `.`（`netlify.toml` で設定済み）
3. 以降は `main` に push すれば自動デプロイ

手作業のドラッグ＆ドロップは不要になる。
